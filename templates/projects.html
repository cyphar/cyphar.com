Projects

Having programmed for most a majority of my life, I have worked on quite a few software projects in that time. The following is a list of the most notable projects I have worked on or contributed to.

dotRush
Very addictive android game. Eat the smaller dots to grow and eat bigger dots. Just don't get dotrush'd.
ninjabot
Modular IRC bot written in Python. Supports dynamic reloading of bot modules and core.
rawline
The small and self-contained line-editing library. Written in less than 1000 lines of ANSI C and can easily be included in any project.
magnesium
(WIP) Explosive messaging -- decentralised, zero-configuration LAN chat system for enterprise.
synge
Very powerful scientific calculation engine. Supports variables, short-cutting conditions and recursive expressions.
perfectgift
A tornado webapp, written by Group 4 at the NCSS Summer School 2014 all-nighter. Get the perfect gift for your friends, with this innovative wishlist app.
epyc
A python templating language (written for perfectgift).
copper
Cu, a C userland.
ttu
Small program which silently converts TCP sockets to Unix sockets for any standard *nix executable.
tic-tac-toe
The classic game, boasting a heuristic AI. Supports zero, one or two players.
ascii-snake
Remake of the Nokia snake game for *nix consoles, with several compile-time configuration options.
moss
A simple command-line file viewer. Supports regex searching, multiple files, switching to $EDITOR and file reloading.
